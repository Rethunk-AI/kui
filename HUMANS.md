# Setup & Operations

User and operator setup guides.

## Prerequisites

- **Go** 1.18+
- **Make**

## Build

```bash
make build
```

## Running

```bash
make run
```

## Development

```bash
make clean
make build
make run
```

## Testing

```bash
make test      # Run tests
make coverage  # Generate coverage reports
```
