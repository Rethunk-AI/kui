# KUI

Web-based KVM User Interface for VM lifecycle, provisioning, and monitoring. Connects to libvirt hosts with a browser-based UI.

**Features:** Admin & user guides, systemd deployment, TLS/reverse proxy support, runtime prefix mode, full test coverage.
